# 📡 NetReaper v1.3 (Advanced Wi-Fi Security Tool)

**NetReaper** — это автономный программно-аппаратный комплекс для проведения комплексного аудита безопасности беспроводных сетей на частоте 2.4 ГГц, построенный на базе микроконтроллера ESP8266. Проект написан на C++ и использует низкоуровневые возможности официального SDK для перехвата и инъекции сырых пакетов в радиоэфир.

---

## ⚡ Полный функционал версии 1.3 (Features)

* **💀 Умное глушение (Deauth Attack)**: Отправка сырых 26-байтных кадров деаутентификации для проверки устойчивости устройств к разрыву соединения.
* **🛡️ Динамический WhiteList**: Система исключений, предотвращающая отключение ваших собственных девайсов. Количество активных слотов динамически расширяется от 1 до 5 прямо через веб-панель.
* **🛰️ Настраиваемый Beacon Spam**: Циклический генератор пакетов-маяков. Позволяет выбрать количество транслируемых сетей (0–5) и вручную задать индивидуальное имя (SSID) для каждой из них через интерфейс.
* **🕵️ Сниффер клиентов (Client Scan)**: Модуль пассивного перехвата пакетов на канале. Переводит чип в режим радиоанализатора на 15 секунд для сбора MAC-адресов устройств, подключенных к целевому роутеру.
* **🌐 Полная кастомизация сети и IP**: Возможность переопределить SSID, пароль и IP-адрес шлюза панели управления (например, `192.168.4.1`, `10.10.10.1`) с сохранением всех данных в EEPROM.
* **🧲 Captive Portal**: Встроенный DNS-сервер автоматически перехватывает трафик подключившихся пользователей и мгновенно выводит всплывающее окно с веб-интерфейсом управления.

## 💻 Краткое руководство пользователя

1. Запитайте плату NetReaper (от USB-порта компьютера, повербанка или аккумулятора).
2. Подключитесь к Wi-Fi сети (по умолчанию: `NetReaper`, пароль: `password123`). На смартфоне сработает Captive Portal и автоматически откроет интерфейс.
3. **Модули**:
   * Нажмите **«Сканировать Роутеры»** для обнаружения целей вокруг.
   * Кнопка **Deauth** активирует атаку на выбранную точку доступа.
   * Кнопка **Clients** запустит 15-секундный перехват MAC-адресов подключенных девайсов.
   * Кнопка **«Запустить Beacon Spam»** поднимет в эфире фейковые сети, которые вы настроили.
4. **Вкладка «Настройки»**:
   * Изменение IP-адреса, имени и пароля самой платы (с автоперезагрузкой).
   * Добавление кастомных имен для Beacon Spam.
   * Управление размером и содержимым Белого Списка (WhiteList).
