# 📡 NetReaper v1.2

**NetReaper** — это автономный программно-аппаратный комплекс для проведения аудита безопасности беспроводных сетей на базе микроконтроллера ESP8266. Устройство сканирует радиоэфир на частоте 2.4 ГГц, обнаруживает точки доступа, перехватывает MAC-адреса подключенных к ним клиентов и позволяет проводить точечные или массовые тесты на деаутентификацию.

Код версии 1.2 полностью оптимизирован для работы с низкоуровневым SDK ESP8266 и энергонезависимой памятью.

---

## ⚡ Что нового в версии 1.2 (Changelog)
* **Сканер клиентов (Client Sniffer)**: Добавлен модуль перехвата сетевых пакетов на выбранном канале. Позволяет увидеть MAC-адреса активных устройств, подключенных к целевому роутеру.
* **Генератор ложных сетей (Beacon Spam)**: Модуль циклической генерации пакетов-маяков. Транслирует в эфир до 10 независимых фейковых Wi-Fi сетей с уникальными MAC-адресами для маскировки или стресс-тестирования клиентских устройств.
* **Кастомный IP-адрес панели**: Возможность динамически изменять IP-адрес шлюза (например, с `192.168.4.1` на `10.10.10.1`) прямо из меню настроек с сохранением в EEPROM.
* **Captive Portal**: Встроенный DNS-сервер автоматически перенаправляет любой трафик подключившихся пользователей на локальную страницу управления.

## 💻 Инструкция по эксплуатации (Usage)

1. Запитайте плату NetReaper через порт USB.
2. Подключитесь к Wi-Fi сети устройства (по умолчанию: `NetReaper`, пароль: `password123`). На смартфоне автоматически откроется окно авторизации.
3. **Главный экран**:
   * Нажмите **«Сканировать Роутеры»**. В таблице появятся доступные цели.
   * Кнопка **Deauth** запускает классическое глушение выбранного роутера.
   * Кнопка **Clients** переводит плату в режим пассивного сниффера на 15 секунд. По завершении под таблицей отобразится список MAC-адресов пойманных клиентов.
   * Кнопка **«Запустить Beacon Spam»** активирует генерацию фейковых сетей во всем диапазоне 2.4 ГГц.
4. **Настройки**:
   * Позволяют переопределить SSID/пароль сети NetReaper.
   * Позволяют задать новый IP-адрес устройства (после сохранения плата автоматически перезагрузится).
   * Позволяют настроить до 5 MAC-адресов в **WhiteList** (устройства-исключения).
