# 📡 NetReaper (ESP8266 Wi-Fi Deauther)

**NetReaper** — это кастомный инструмент для аудита беспроводных сетей на базе микроконтроллера ESP8266. Устройство сканирует радиоэфир на частоте 2.4 ГГц, находит доступные точки доступа и позволяет отправлять сырые пакеты деаутентификации (Deauth Frames) для тестирования стабильности и безопасности Wi-Fi сетей.

Проект написан «с нуля» на C++ в среде Arduino IDE без использования сторонних прошивок. Он сочетает низкоуровневую работу с радиомодулем и веб-интерфейс, развернутый прямо на чипе.

---

## ⚡ Особенности (Features)
* **Автономность**: Создает собственную зашифрованную Wi-Fi сеть для управления.
* **Встроенный сканер**: Автоматически сканирует каналы 1–13 и находит скрытые (Hidden) SSID.
* **Удобный веб-интерфейс**: Адаптивная темная HTML/CSS панель для управления с любого смартфона или ПК.
* **Оптимизированный движок атак**: Балансирует работу низкоуровневой отправки пакетов и веб-сервера, предотвращая обрывы связи с панелью управления.

---

## 🛠️ Как это работает (Принцип действия)
Устройство использует «смешанный» режим работы радиомодуля (`wifi_promiscuous_enable`). При запуске атаки NetReaper формирует поддельный 26-байтный широковещательный кадр отключения от имени целевого роутера:

```text
[ Тип: Deauth ] -> [ Получатель: FF:FF:FF:FF:FF:FF ] -> [ Отправитель: MAC Роутера ]
```
Устройства, подключенные к роутеру, получают этот пакет в незашифрованном виде, считают его легитимным приказом от точки доступа и мгновенно разрывают соединение.

## 🚀 Быстрый старт (Installation)

### 1. Подготовка среды
1. Установите **Arduino IDE**.
2. В настройках IDE в поле *«Дополнительные ссылки для Менеджера плат»* добавьте:
   `http://esp8266.com`
3. Через *Менеджер плат* найдите и установите платформу **esp8266**.

### 2. Загрузка прошивки
1. Откройте скетч `Deauther.ino` в Arduino IDE.
2. В меню *Инструменты -> Плата* выберите вашу модель (например, **NodeMCU 1.0** или **Wemos D1 Mini**).
3. Подключите плату по USB, выберите соответствующий COM-порт и нажмите **Загрузить**.

---

## 💻 Использование (Usage)

1. Подайте питание на плату (от USB, повербанка или аккумулятора).
2. Подключитесь с телефона к Wi-Fi сети устройства:
   * **SSID:** `NetReaper`
   * **Пароль:** `password123`
3. Откройте браузер и перейдите по адресу: **`http://192.168.4.1`**
4. Нажмите **«Сканировать эфир»**, выберите цель и нажмите **«Атака»**. Для остановки нажмите красную кнопку **«ОСТАНОВИТЬ»**.
