# 📡 NetReaper v1.1

**NetReaper** — это автономный инструмент для аудита безопасности беспроводных сетей на базе микроконтроллера ESP8266. Устройство сканирует радиоэфир на частоте 2.4 ГГц, обнаруживает точки доступа и отправляет низкоуровневые пакеты деаутентификации (Deauth Frames) для проверки устойчивости Wi-Fi сетей к атакам.

В версии 1.1 проект получил полноценную панель настроек, энергонезависимую память и автоматическую маршрутизацию трафика.

---

## ✨ Что нового в версии 1.1 (New Features)
* **Captive Portal**: Встроенный DNS-сервер автоматически открывает интерфейс управления при подключении к Wi-Fi сети устройства.
* **WhiteList (Белый список)**: Возможность добавить до 5 MAC-адресов устройств-исключений, которые не будут затронуты во время проведения тестов.
* **Энергонезависимая память (EEPROM)**: Настройки имени сети, пароля и белого списка сохраняются в памяти чипа и не сбрасываются при отключении питания.
* **Меню настроек**: Полноценный веб-интерфейс для изменения параметров точки доступа и управления списком исключений.

---

## 🛠️ Архитектура проекта
Программа работает в гибридном режиме. Для отправки сырых пакетов используется режим `wifi_promiscuous_enable`. Чтобы веб-интерфейс оставался доступным во время атаки, отправка пакетов чередуется с короткими паузами для обработки сетевого стека встроенного веб-сервера.

## 🚀 Быстрый старт (Installation)

### 1. Подготовка среды
1. Установите **Arduino IDE**.
2. В настройках IDE в поле *«Дополнительные ссылки для Менеджера плат»* добавьте:
   `http://esp8266.com`
3. Через *Менеджер плат* найдите и установите платформу **esp8266**.

### 2. Загрузка кода
1. Откройте файл `Deauther.ino` (код версии 1.1) в Arduino IDE.
2. В меню *Инструменты -> Плата* выберите вашу плату (например, **NodeMCU 1.0** или **Wemos D1 Mini**).
3. Подключите устройство по USB, выберите нужный COM-порт и нажмите **Загрузить** (Upload).

## 💻 Инструкция по эксплуатации (Usage)

1. Подключите NetReaper к любому источнику питания (повербанк, USB-порт компьютера или адаптер питания).
2. Откройте Wi-Fi на телефоне или ПК и найдите сеть по умолчанию:
   * **SSID:** `NetReaper`
   * **Пароль:** `password123`
3. Подключитесь к сети. Операционная система сама предложит войти в сеть и откроет интерфейс управления. Если этого не произошло, введите в браузере любой адрес (например, `deauth.me` или `192.168.4.1`).
4. **Управление**:
   * Нажмите **«Сканировать эфир»** на главной странице, выберите целевую сеть из таблицы и нажмите **«Атака»**.
   * Для прекращения отправки пакетов нажмите красную кнопку **«ОСТАНОВИТЬ»**.
   * Перейдите во вкладку **«Настройки»**, чтобы сменить имя/пароль скрытой сети NetReaper или внести MAC-адреса своих устройств в WhiteList.
